# 新闻搜索引擎
系统开发环境：Eclipse<br>
开发语言：Java<br>
数据库：MySQL<br>
运行平台： Windows 10<br>
相关技术：Spring+SpringMVC+Solr<br>

**爬虫**（有待优化）<br>
开发环境：PyCharm<br>
开发语言：Python<br>
数据库：MySQL<br>
运行平台： Windows 10<br>
数据来源：腾讯体育、新浪体育、搜狐体育

说明：一个基于Solr的搜索引擎，由Spring+SpringMVC框架搭建，通过Solr服务器对MySql数据库表进行索引和检索，数据库中数据爬取自于上述三个网站的部分内容，有待优化。
